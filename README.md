# C-Homes: Affordable Housing for NYSC in Ibadan

C-Homes is a modern React-based property search platform dedicated to helping National Youth Service Corps (NYSC) members find verified and affordable accommodation in Ibadan, Oyo State.

## 🌟 Features

- **Verified Listings**: Every apartment is checked for quality and safety.
- **Corper-Friendly Prices**: Housing tailored to NYSC allowance budgets.
- **Location Filtering**: Browse apartments by Local Government Area (LGA) and specific areas in Ibadan.
- **Direct Inquiry**: Connect directly with agents via WhatsApp for seamless inquiries.

## 🚀 Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, lucide-react (icons)
- **UI Components**: shadcn/ui (Radix UI)
- **Data Management**: TanStack Query (React Query)

## 🛠️ Getting Started

To run the project locally, follow these steps:

1.  **Clone the repository**:
    ```sh
    git clone https://github.com/TosinISOGUN/c-homes.git
    cd c-homes
    ```

2.  **Install dependencies**:
    ```sh
    npm install
    # or if you use bun
    bun install
    ```

3.  **Start the development server**:
    ```sh
    npm run dev
    ```

4.  **Build for production**:
    ```sh
    npm run build
    ```

## 📂 Project Structure

- `src/pages`: Main views (Home, Listings, About, Contact, etc.).
- `src/components`: Reusable UI elements and layout components.
- `src/data`: Mock data for apartment listings.
- `src/lib`: Utility functions and shared logic.
